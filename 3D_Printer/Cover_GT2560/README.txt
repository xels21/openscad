# Summary

Cover GT 2560 for GEEETECH i3. Glue is not required

# Print Settings

Printer: Geeetech I3
Rafts: Doesn't Matter
Supports: No
Infill: 20
Filament_brand: Tianse
Filament_color: White or Black
Filament_material: PLA

Notes: 
This project has two purposes: the first is to create a component for the GEEETECH i3 printer that can be updated over time by replacing its parts without having to reprint the entire piece; the second objective is to print the entire project in several phases in order to constantly monitor the printing process.
