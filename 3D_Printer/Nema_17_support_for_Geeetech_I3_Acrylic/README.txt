# Summary

This is a support for the extruder (Nema17 motor).
It fit perfectly with the Geeetech Prusa I3 Acrylic frame.
This is usefull if you want convert your Prusa from direct drive to bowden.
I also updated an E3DV6 support for the metal X carriage, search at "Geeetech E3DV6 conversion"
