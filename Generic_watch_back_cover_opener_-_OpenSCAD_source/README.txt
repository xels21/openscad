# Summary

I needed to replace the battery of my watch and I didn't want to damage it therefore I've created this back cover opener. OpenSCAD source is included.

**Update 10-02-2018:** Added 38, 39, 40mm versions in both 3 or 6 pin combinations.
Both the OpenSCAD and STL files are available.
Item in the picture is the 40mm 6pin version.
